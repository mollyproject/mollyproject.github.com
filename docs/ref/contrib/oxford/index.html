University of OxfordÂ¶

The University of Oxford deployment is at http://m.ox.ac.uk/ and goes by the name Mobile Oxford.

Apart from local configuration and such things as API keys, the entirity of the Oxford deployment is kept in the Molly source tree (under molly.contrib.oxford) in order that it may be used for inspiration or as a reference at other institutions.

Mobile Oxford currently provides a variety of Molly applications.
