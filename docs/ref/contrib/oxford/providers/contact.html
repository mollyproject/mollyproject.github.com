Contact providers¶

class molly.contrib.oxford.providers.ContactProvider¶

This provider makes use of an IP-restricted service and is thus of little use beyond providing insight into how to develop a contact provider.

class molly.contrib.oxford.providers.ScrapingContactProvider¶

This provider performs requests against the University’s contact search, extracting the results from the returned HTML.

As results from the main site are already paginated in groups of ten, this provider handles its own pagination.
