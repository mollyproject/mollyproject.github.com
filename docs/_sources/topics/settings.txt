The Molly settings file
=======================

.. module:: settings

The Molly settings file is much the same as that used in any other Django
project. If you are not familiar with Django settings files you are
advised to read `their documentation
<http://docs.djangoproject.com/en/1.1/topics/settings/>`_ first.

In addition to those documented in Django, the following are available:

.. data:: APPLICATIONS

   A list of :class:`~molly.conf.settings.Application` objects.
