Testing¶

Browser testing¶

Emulators for various platforms can be found at the following locations:

Android
Cross-platform emulator. Requires a bit of fiddling to get running on Linux x86_64 (but certainly possible).
Apple iOS
Distributed as part of the iOS SDK. Requires Mac OS X.
Opera Mini
A Java Applet (with a rather small screen resolution).
Opera Mobile
Cross-platform emulator.
